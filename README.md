Project is created with:

- NPM version: 8.3.1
- FE - React - Typescript
- Storybook

## Setup

```
# Install server dependencies
npm install

# Install client dependencies
cd client
npm install

# Run both Express & React from root
cd ..
npm run dev

# Build for production
cd client
npm run build

# Run storybook
cd client
npm run storybook

# To Test
cd client
npm run test
```
